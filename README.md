# Install-Msi.ps1
- Installs an MSI package through the usage of PowerShell 5.1+.
## Usage
```powershell
Install-Msi -Path "C:\Path\To\Installer.msi" -Arguments "/quiet"
```
## Parameters
- `Path`: The file path to the MSI installer package.
- `Arguments`: Optional arguments to pass to the MSI installer (e.g., `/quiet` for silent installation).
